
#### 형변환 (Casting)  
  
***변수 또는 상수의 타입을 다른 타입으로 변환하는것 why?다른 타입끼리는 연산x so 연산전에 타입을 일치시킴***  
  
  
1)방법 : 변환을 원하는 변수나 리터럴 앞에 변환하고자 하는 타입을 괄호안에 붙인다.  
```(타입)피연산자  
  double d = 26.5;  
  int score = (int)d; 또는 int score = (int)26.5;
  ```  
형변환을 한 score는 int값으로 출력, 피연산자였던 d는 그대로 출력  
  
기본형끼리는 boolean을 제외하고 모두 변환 가능.  
  
###### 1.boolean을 제외한 나머지 기본형은 서로 형변환 가능.  
###### 2.기본형 참조형 서로 형변환 불가능.  
###### 3.다른 타입의 변수간의 연산은 형변환이 원칙.값의 범위가 작은것->큰것은 생략 가능  
  
  
  
## chapter03- 연산자  
연산자 : 연산을 수행하는 기호  
피연산자 : 연산을 당하는 대상 (변수,상수,리터럴,수식)  
  
***연산자는 피연산자로 연산을 수행하면 항상 결과값을 반환***  
  
#### 1.2 식과 대입 연산자  
  
```4*x+3; // ;을 사용해야 문장이 성립```  
  
  
```4 * x + 3; // x의 값이 5로 초기화 되었을 때 결과값은 반환되나 사라진다.  
  
  so 대입 연산자를 사용해서 값을 담아주어야 함.  
  y = 4 * x + 3;  //식의 결과가 변수 y에 저장됨.  
  ```
